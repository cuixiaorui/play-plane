## play-plane

在 vue3 中采用 Functional Core, Imperative Shell 模式来实现的打飞机游戏

用来探索 Vue3 中如何做到 UI 层和业务逻辑层分离架构的实现

其目的是为了测试重度前端项目

![](https://images-1252602850.cos.ap-beijing.myqcloud.com/20220414150701.png)


## 技术栈

- typescript
- vue3 setup
- vitest
- custom renderer
- pixi.js

## 开始

```
pnpm i 

pnpm dev
```
